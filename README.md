# MacApp Support Website

这是MacApp的支持网站，用于App Store审核。

## 网站内容

- **首页** (`index.html`): App介绍和功能展示
- **隐私政策** (`privacy.html`): 详细的隐私政策说明
- **使用条款** (`terms.html`): 应用使用条款和条件
- **支持页面** (`support.html`): 联系方式和常见问题

## 技术栈

- 纯静态HTML5
- CSS3（响应式设计）
- 原生JavaScript

## 访问地址

网站已部署到GitHub Pages：https://justfavor.github.io/macapp-support/

## 本地预览

直接在浏览器中打开`index.html`文件即可预览。
