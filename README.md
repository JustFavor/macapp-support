# App Support - Mac应用支持中心

这是App Support的官方网站仓库，用于展示和提供Mac应用程序的支持信息。

## 网站结构

```
├── index.html          # 首页 - 展示所有应用
├── apps/               # 应用详情页面
│   └── rightclick-master.html  # 右键大师详情页
├── privacy.html        # 隐私政策
├── terms.html          # 使用条款
├── support.html        # 联系我们
├── style.css           # 样式文件
└── script.js           # 脚本文件
```

## 特色

- 🎨 现代简洁的设计风格
- 📱 响应式布局，支持移动设备
- 🚀 多应用支持架构
- 🔒 注重隐私保护
- ⚡ 原生支持 Apple M系列和 Intel 芯片

## 当前应用

### 右键大师
强大的Mac右键菜单增强工具，提供新建文件、常用目录、发送/复制文件等功能。

## 本地开发

1. 克隆仓库
```bash
git clone https://github.com/JustFavor/macapp-support.git
```

2. 在浏览器中打开 `index.html`

## 部署

网站通过 GitHub Pages 自动部署。推送到 main 分支后，网站将自动更新。
